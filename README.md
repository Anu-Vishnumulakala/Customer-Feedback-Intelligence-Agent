# Customer-Feedback-Intelligence-Agent
An AI-powered system that transforms raw customer feedback into structured insights, helping businesses understand sentiment, urgency, and key improvement areas.
## Project Overview
Customer feedback is often unstructured and difficult to analyze at scale.
This project uses AI to automatically extract meaningful insights from customer reviews, surveys, or support messages.
The system identifies:
- Customer sentiment
- Main topics
- Urgency level
- Recommended business actions
This enables faster, data-driven decision making for marketing and product teams.
## Key Features
- Sentiment Analysis (Positive / Neutral / Negative)
- Topic Detection (Product, Pricing, Support, UX, etc.)
- Urgency Classification
- Actionable Business Recommendations
- Summary Dashboard
- Interactive Streamlit Interface
## How It Works
 - User uploads a CSV file with customer feedback
 - Each feedback entry is analyzed using an LLM
 - The system extracts structured insights
 - Results are aggregated into a business summary
 - Insights are displayed in a dashboard
